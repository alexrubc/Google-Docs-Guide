# Text formatting
## Changing a text style
1. In Google document, select the text you want to change.
2. Click **Format > Text**.
3. Choose one of the following:

| Text style | Sample|
| ----------- | ----------- |
| Bold | **to process a text like this** |
| Italic | *to process a text like this* |
| Underline | <u>to process a text like this </u> |
| Strikethrough | <strike>to process a text like this</strike> |
| Superscript | <sup>to process a text like this</sup> |
| Subscript | <sub>to process a text like this</sub> |

## Changing a text size
1. In Google document, select the text you want to change.
2. Type the desired value into the **Font size** field on the toolbar.  ![scr](img/textsize1.png)

## Changing a text capitalisation
In Google document, select the text you want to change and click **Format > Text > Capitalisation**, then choose one of the following:  

- **UPPERCASE**
- **lowercase**
- **Title Case**
